# Tableau-HR-dashboard

A collection of Tableau dashboards and business intelligence use cases, focusing on KPI-driven insights, interactive visual analytics, and BI best practices

Developed end-to-end Tableau dashboards using real-world business datasets, translating raw data into executive-ready visual analytics with clear KPI definitions and actionable insights.

Applied best-practice data modeling and calculated fields (LOD expressions, table calculations, parameters) to enable dynamic slicing, drill-down analysis, and performance-optimized dashboards.

Designed interactive dashboards following BI and data visualization principles (clarity, hierarchy, color theory), improving insight discoverability and decision-making usability.

Implemented advanced Tableau features including filters, actions, tooltips, and dashboard navigation to support self-service analytics for non-technical stakeholders.

Documented dashboard logic, KPI assumptions, and visualization choices, demonstrating strong analytics communication and BI governance practices.
